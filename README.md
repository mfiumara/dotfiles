# dotfiles

On macOS, install dependencies with:

```bash
./scripts/setup_macos.sh
```

This installs the following Homebrew packages when missing:

* nvim
* node
* ripgrep
* tmux
* code-minimap
* python3

## Installation

```bash
./scripts/setup_macos.sh
./scripts/setup_nvim.sh
./scripts/setup_tmux.sh
```
